# Books

Technical books written for the cloud computing and IT service management space: two self-published AWS certification guides and two ghostwritten titles.

## AWS Certification Guides (Amazon KDP)

Published under the pen name Allison Cope, these two exam guides simplify complex cloud computing concepts for the AWS Solutions Architect Associate certification, the world's #1 cloud certification. Each topic is deconstructed with exam-focused content, real use-case scenarios, architectural diagrams, and cheat sheets, so readers can translate knowledge into real-world application.

### AWS Certified Solutions Architect Associate (SAA-C02) Study Guide

A comprehensive study guide covering AWS global infrastructure, architectural principles, and core deployment, operations, and security concepts. Written to be technically deep without losing clarity, for both newcomers to cloud computing and existing AWS professionals.

[View on Amazon](https://www.amazon.com/dp/B093RRTYKF)

### AWS Solutions Architect Associate C02 Exam Guide (2020-21)

A last-minute study guide that walks through exam-specific questions using real use-case scenarios and architectural diagrams. Covers storage, databases, compute, multi-tier applications, VPC security, encryption, and network architectures. No prior AWS experience needed.

[View on Amazon](https://www.amazon.com/dp/B085K78CZH)

## Ghostwritten Books

### Information Technology Infrastructure Library (ITIL) 4 Book

Served as the ghostwriter for a comprehensive guide to ITIL 4, the leading framework for IT service management, commissioned by an international client. The work involved extensive research and close collaboration with the client to ensure clarity and relevance throughout. Each chapter equips IT professionals with strategies and tools to improve service quality, boost efficiency, and promote innovation, with a focus on aligning IT services with business goals.

### Ebook on Global Project Management for Remote Software Teams

Ghostwrote an ebook for a Germany-based SaaS company on managing distributed software teams: asynchronous workflows, cross-cultural alignment, adapting Agile for remote delivery, and scaling global teams without losing velocity. Content withheld under ghostwriting confidentiality.
